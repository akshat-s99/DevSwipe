================================================================================
                              ANIMATIONS ADDED
================================================================================

ANIMATION 1: fadeInUp
─────────────────────
Trigger: Page load
Duration: 0.6s
Easing: cubic-bezier(0.34, 1.56, 0.64, 1) [elastic bounce]
Effect: Card slides up from 30px below with fade in

Code:
  @keyframes fadeInUp {
    from {
      opacity: 0;
      transform: translateY(30px);
    }
    to {
      opacity: 1;
      transform: translateY(0);
    }
  }

ANIMATION 2: moveGrid
────────────────────
Trigger: Continuous loop
Duration: 20s
Timing: Linear infinite
Effect: Subtle grid pattern moves in background

Code:
  @keyframes moveGrid {
    0% {
      transform: translate(0, 0);
    }
    100% {
      transform: translate(50px, 50px);
    }
  }

ANIMATION 3: Button Hover Lift
──────────────────────────────
Trigger: Mouse over button
Duration: 0.3s
Effect: Button lifts up, shadow grows

Code:
  &:hover:not(:disabled) {
    transform: translateY(-2px);
    box-shadow: 0 12px 24px rgba(34, 197, 94, 0.3);
  }

ANIMATION 4: slideDown (Alerts)
───────────────────────────────
Trigger: Error appears
Duration: 0.3s
Effect: Error message slides down from top

Code:
  @keyframes slideDown {
    from {
      opacity: 0;
      transform: translateY(-12px);
    }
    to {
      opacity: 1;
      transform: translateY(0);
    }
  }
