================================================================================
                        REQUIRED BACKEND
================================================================================

The frontend expects the following backend API endpoints:

Authentication:
  POST /api/auth/register
  POST /api/auth/login

User Profiles:
  GET /api/profiles
  GET /api/users/:userId

Swiping:
  POST /api/swipes

Matches:
  GET /api/matches

Messaging:
  GET /api/messages/:matchId
  POST /api/messages

Real-time:
  Socket.io server on port 8000
  Events: joinChat, sendMessage, newMessage
